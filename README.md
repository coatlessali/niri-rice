## Dependencies

- niri
- waybar
- wofi
- xwayland-satellite
- kitty
- pulsemixer
- mpv
- bash
- all of the scripts in ~/.local/bin (make them executable)
- the .sounds folder in your home directory
- swaybg
- Inconsolata font
- Fira Sans font

You *will* need to edit your niri config, unless your name is also coincidentally Ali.
